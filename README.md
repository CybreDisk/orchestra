# orchestra

> CybreDisk build system

## Install

```sh
$ git clone --recurse-submodules https://github.com/CybreDisk/orchestra.git
```

## Usage

From the orchestra folder run:

```sh
$ ./make.sh
```

This will create a file system in the `prefix` folder.
